Handsontable Custom Builder v@@version

Usage: hot-builder <command>

Available commands:
  help - Display this help text.
  version - Print the Handsontable version.
  build - Build custom version of Handsontable.

Build Options:
  --input, -i       Input package.json file (default entry point is node_modules/handsontable/package.json).
  --output-dir, -o  Output directory.
  --all, -a         If exists it includes by default all founded modules.
  --add-plugin      Adds specified plugin to build package (eg. --add-plugin=ContextMenu,Autofill or --add-plugin=ContextMenu --add-plugin=Autofill).
  --remove-plugin   Removes specified plugin from build package (eg. --remove-plugin=ContextMenu,Autofill or --remove-plugin=ContextMenu --add-plugin=Autofill).
  --disable-ui      Disable UI
